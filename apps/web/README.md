# Webview

This app is a Vite SPA using TanStack Router (file-based routes), Tailwind CSS v4 (via Vite plugin), and shadcn/ui.

## Commands

- `bun install` - install dependencies
- `bun run dev` - start the local dev server
- `bun run lint` - run ESLint
- `bun run preview` - preview the production bundle

## Routing

- Routes live in `src/routes`
- `src/routes/__root.tsx` defines the app shell/root layout
- `src/routes/index.tsx` defines the `/` route
- `src/routeTree.gen.ts` is generated by TanStack Router (do not edit)

## shadcn/ui

- Config is in `components.json`
- Add new components with `bunx shadcn@latest add <component>`
- Shared utilities are in `src/lib/utils.ts`
